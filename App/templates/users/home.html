Fetal Health Classification

Cardiotocography (CTG) is a valuable technique used to monitor the uterine contractions and fetal heart rate during pregnancy and labor. It involves two key signals: Continuous Fetal Heart Rate (FHR) monitoring and Uterine Activity (UC). This project utilizes both Machine Learning (ML) and Deep Learning (DL) methods for fetal health classification to identify the state of the fetus as Normal, Suspicious, or Pathological.

Methods Used

Deep Learning: This project applies DL-based techniques to process large amounts of CTG data, using Convolutional Neural Networks (CNNs) and Recurrent Neural Networks (RNNs) for feature extraction and time-series analysis. DL methods can handle both standard and non-standard patterns in the data, making them suitable for complex and diverse cases.

Machine Learning: For simpler classification, ML techniques such as Random Forest, Support Vector Machine (SVM), and k-Nearest Neighbors (k-NN) are used to classify the CTG data into three categories: Normal, Suspicious, and Pathological. These methods are effective for standard cases and are less computationally intensive than DL.

Advantages and Benefits

  • Improved Accuracy: Using a combination of ML and DL techniques allows for better detection of subtle abnormalities in fetal health, leading to more accurate predictions.
  • Handling Complex Data: DL techniques can process large volumes of CTG data, capturing intricate patterns that traditional methods may miss.
  • Real-Time Monitoring: Both ML and DL methods enable real-time analysis of fetal health, allowing for timely intervention in critical cases.
  • Scalability: The methods used in this project can be scaled for large datasets, accommodating different hospitals and healthcare settings.
